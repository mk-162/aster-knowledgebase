# Running the event on race day

Your **organiser dashboard** is mission control for the day: it lists your
events, the live participant roster, and any safety alerts, and refreshes
continuously.

## Start the race

For a mass-start race, press **Start** to fire the gun — this sets the official
start time that every rider's clock runs from. If you forget, Watch the Dot will
**auto-start** the race shortly after the scheduled start time so the leaderboard
still has a proper clock. Press Start yourself if you need the gun to the second.

You can't start until a finish line is set, and you can't restart an event that's
already finished or cancelled.

## Watch the field

- See every rider's **status** (registered, started, finished, DNF), bib,
  category, team, current lap and last update time.
- Filter the roster by status.
- Watch the same live map and leaderboard your spectators see.

## Fix problems on individual riders

From the participant list you can:

- **Force finish** a rider (for example, they crossed but their phone was offline).
- **Mark DNF** a rider who has pulled out.
- **Reset to registered** so a rider can start again (clears their previous
  attempt's live state).
- **Clear a stuck tracking session** if a rider's device gets into a bad state.

## Safety alerts

SOS alerts from riders appear on your dashboard with the rider's location, so you
can respond. This has its own guide — see [Handling SOS alerts](handling-sos-alerts.md).

## Message participants

- **Broadcast** a message (with an optional link) to everyone in the event — it
  reaches riders as a notification.
- **Post to the event feed** — updates that registered riders and event followers
  see, and can be notified about.

## Important: once a rider finishes, they're done

When a rider crosses the line, is cut off, or you finish them manually, their
result **freezes** — their dot stops at the finish and their time and position
lock in. Late GPS points from their phone afterwards are ignored, so finished
riders never drift on the map. This keeps your results clean.
