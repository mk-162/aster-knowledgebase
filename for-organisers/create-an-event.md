# Create an event

This is the big-picture walkthrough; each part links to a detailed guide.

## Steps

1. **Choose the host.** Pick which organiser account (or team) hosts the event.
   This is fixed once created.
2. **Pick the event type** — mass-start race, time trial, or group ride /
   adventure. This affects timing and finishing. See
   [Event types explained](event-types.md).
3. **Add the route.** Upload a GPX file; the app reads the distance, elevation
   and shape, and can place your start and finish from it. See
   [Routes, start and finish lines](routes-start-and-finish.md).
4. **Set the start time and time zone.** Always pick the event's local time zone
   explicitly — this is what the race clock and reminders use.
5. **Set a cut-off (optional).** A wall-clock deadline after which riders still
   out are closed out automatically. You choose whether they're marked DNF or
   finished.
6. **Choose privacy** — public, unlisted or private. See
   [Public, unlisted and private events](../for-spectators/public-unlisted-private-events.md).
7. **Configure registration and categories.** Open or approval-based joining,
   capacity, and your racing categories. See
   [Registration and categories](registration-and-categories.md).
8. **Publish.** Your event page goes live at its own web address.

## Important

- **Always set the correct time zone.** The start time you enter is interpreted in
  the event's local zone, and the race clock and reminders depend on it. Don't
  leave it on a default.
- **A finish line is required to start the race.** You can adjust the
  auto-detected start/finish at any time before the event — see
  [Routes, start and finish lines](routes-start-and-finish.md).

## After creating

You can edit most settings up until the event starts. Some settings lock once
riders have registered (for example, switching registration into approval mode
when people have already joined). The event then moves through its lifecycle:

| Status | Meaning |
|---|---|
| **Upcoming** | Created, not started |
| **Live** | Race is running |
| **Finished** | Ended; results can be published |
| **Cancelled** | Called off |
| **Archived** | Historical, read-only |
