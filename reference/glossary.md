# Glossary

**Abandon** — Ending an event ride before the finish; recorded as DNF.

**Bib** — A rider's number for an event, shown on the leaderboard.

**Category** — A group within an event (e.g. "Pro Women"); results are ranked
within each category.

**Checkpoint** — A marked point along a route (an aid station, a turn, a
waypoint), used especially on point-to-point and time-trial courses.

**Course** — A reusable route template — its shape, start/finish lines and feed
stations — that an organiser can apply to several events at once.

**Crash detection** — The app's automatic detection of a sudden impact, which can
raise an SOS for you if you don't cancel it.

**Cut-off** — A deadline set by the organiser; riders still out past it are closed
out automatically.

**DNF** — "Did not finish": started but didn't complete the event.

**DNS** — "Did not start": registered but never started.

**Dot** — Your live position on the map.

**Event** — An organised race, group ride or adventure you register for.

**Feed station** — A support point (water, food, aid) marked on the route.

**Festival** — A group of related races run under one banner (for example several
distances at one weekend).

**Free ride** — A personal ride you track outside any event; private by default.

**Grand départ** — A point-to-point adventure where riders set off individually.

**Gun** — The official start of a mass-start race; everyone's clock starts here.

**Handle** — Your @username, used for links and @mentions.

**Lap** — One circuit of a looped course; multi-lap events count laps as riders
pass the line.

**Leaderboard** — The live standings for an event (the "Riders" tab).

**Mass-start race** — An event where everyone starts together on the gun.

**Moving time** — Your elapsed time minus any time you were paused.

**Off route** — When a rider has left the event's course.

**Organiser account** — The brand (club, series or company) that hosts events;
can have several admins.

**Ping** — A single GPS point recording your position at one moment.

**Private event** — Hidden and locked; requires signing in and the organiser's
access code to view.

**Public event** — Listed in Discover and watchable by anyone.

**Share link (watch link)** — A private link a rider creates so others can watch
their free ride live, with no account needed.

**SOS** — An emergency alert a rider raises (or crash detection raises), sharing
their live location with watchers, the organiser and emergency contacts.

**Spectator** — Anyone watching an event or ride; usually needs no account.

**Split time** — A rider's time at a checkpoint, used to compare riders
section by section (mainly time trials).

**Start line / finish line** — The points where a ride's timing begins and ends,
each with a detection radius.

**Stoppage** — When a rider has been stationary for a while; can trigger a
"stopped" notification to watchers.

**Time trial** — An event where riders start individually and are timed on their
own clock.

**Trail** — The line on the map showing where a rider has been.

**Unlisted event** — Not listed publicly, but watchable by anyone with the link.

**Verified** — A badge showing an organiser or team has been confirmed by the
Aster team.

**Wave** — A category's own start time, so groups set off in stages rather than
all at once.
